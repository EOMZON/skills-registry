# Frontend Design

Create distinctive, production-grade frontend interfaces with clear visual direction.

## Use When

- 需要做网站、落地页、Dashboard 或内容型页面
- 需要把设计方向和前端实现一起推进

## Input

- `goal`：你要做的页面或界面
- `constraints`：风格、技术栈、设备、交付要求

## Returns

- 设计方向
- 前端实现
- 体验和验收建议

## Notes

- 公开版保留调用契约和设计原则
- 上游私有源里的本地模板路径和团队偏好已移除
