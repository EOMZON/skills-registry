# Homepage Overlay

Use this reference when the audited surface includes a homepage, landing page, or board index.

## Goal

Check whether the first screen helps a newcomer understand identity, choose a path, and trust the next step.

## Questions

1. Identity: can a newcomer tell what this is, who it is for, and why it matters?
2. Freshness: does the page reflect the current product and current priorities?
3. Navigation: is the next step obvious and well-labeled?
4. Interruption: is the user being blocked before understanding value?
5. Durability: does the page still make sense if the styling novelty is stripped away?

## Output shape

```text
Identity: pass / risky / fail
Freshness: pass / risky / fail
Navigation: pass / risky / fail
Interruption: pass / risky / fail
Durability: pass / risky / fail
```
