# Zon Minimal Editorial

把内容型页面做成可读、可回看、可归档的出版物式体验。它是一个明确的 editorial family，不是所有网站的默认答案。

## Use When

- 需要为网站、报告、作品集、博客或档案页建立统一的 editorial house style
- 需要黑白克制、内容优先的视觉方向
- 需要减少卡片墙和装饰性 UI，让层级、节奏和排版承担主要表达

## Input

- `artifact_type`：要处理的页面或产物类型
- `content_shape`：现有结构、页面问题或目标阅读体验

## Returns

- 适配判断和页面原型方向
- typography / spacing / hierarchy 约束
- 可复用的 house style 执行建议

## Notes

- 公开版保留风格原则和落地规则
- 私有模板路径与项目内默认资源已抽象掉
- 如果项目已经有更适合的原生语言，例如更现代的 product shell、目录型系统或 richer atmospheric UI，应优先保留原生语言，而不是强行收口成 editorial。
