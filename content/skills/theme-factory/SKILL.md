# Theme Factory

给一个已有产物补上一套完整主题，而不是只改几个颜色或字体。

## Use When

- 需要为 HTML、文档、幻灯或报告选一套统一主题
- 需要快速比较不同视觉方向
- 需要把主题 tokens 应用到真实交付物里

## Input

- `artifact_type`：要处理的产物类型
- `style_goal`：想要的气质、受众与使用场景

## Returns

- 可选主题方向或自定义主题方案
- 颜色、字体与视觉 tokens
- 应用到成品时的样式执行建议

## Notes

- 公开版保留主题选择和应用逻辑
- 私有素材库、团队默认模板和个人路径约定已移除
